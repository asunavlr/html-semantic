Guia de Instalação

Requisitos Instalação Configuração Verificação

Requisitos do Sistema

Para instalar o DevFramework, você precisa de Node.js 20+ e npm 10+. Sistemas operacionais suportados: Linux, macOS e Windows 10+.

Instalação

Execute o comando npm install devframework no terminal do seu projeto. Para instalação global, adicione a flag -g.

Configuração Inicial

Após a instalação, crie o arquivo dev.config.js na raiz do projeto. O framework oferece um assistente de configuração via npx devframework init.

Verificação

Para verificar se a instalação foi bem-sucedida, execute npx devframework --version. A saída esperada é o número da versão instalada.
